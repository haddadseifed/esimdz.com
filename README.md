# eSIM DZ Web Site - Professional Version

هذا المستودع يحتوي على النسخة الاحترافية والمحسّنة لموقع **eSIM DZ** الثابت.

## محتويات المشروع

*   **`index.html`**: الصفحة الرئيسية للموقع.
*   **`assets/`**: مجلد يحتوي على ملفات CSS و JavaScript والأصول الأخرى.

## النشر (Deployment)

هذا الموقع هو موقع ثابت (Static Website) ويمكن نشره بسهولة باستخدام خدمات استضافة المواقع الثابتة مثل:

*   **GitHub Pages**: إذا قمت بإنشاء مستودع جديد ورفعت هذه الملفات إليه، يمكنك تفعيل خدمة GitHub Pages لنشر الموقع مباشرة.
*   **Netlify / Vercel**: خدمات استضافة سريعة ومجانية للمواقع الثابتة.

## ملاحظات هامة

*   تم تعديل مسارات ملفات الأصول (CSS/JS) في ملف `index.html` لتكون نسبية، مما يضمن عمل الموقع بشكل صحيح عند نشره على أي نطاق فرعي أو مجلد.
*   تم حذف أي كود أو مسارات خاصة ببيئة التطوير (Manus) لضمان نظافة الكود واحترافيته.
